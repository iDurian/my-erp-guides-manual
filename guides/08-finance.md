# 财务逻辑

本模块用于说明财务单据如何与业务链路关联，以及如何查看应收、应付和资金风险。

配套截图指引见：[财务流程：发票、收款、付款](./finance-invoices-payments/README.md)。

退款、费用、报销和财务调整的配套截图指引见：[例外业务与专题报表](./exceptions-reports/README.md)。

## 财务链路入口

- 出库单产生应收链路。
- 入库单产生应付链路。
- 销售发票用于登记客户正式开票金额。
- 采购发票用于登记供应商正式开票金额。
- 收款单登记客户实际到账。
- 付款单登记供应商实际付款。

## 应收流程

应收流程通常包括：

```text
库存出库单 -> 销售发票 -> 收款单 -> 应收看板 / 应收账龄 / 资金流水
```

讲解重点：

- 出库单是应收核对的重要来源。
- 销售发票用于确认正式开票金额。
- 收款单记录实际到账。
- 应收看板展示已收、待收和逾期风险。

## 应付流程

应付流程通常包括：

```text
采购入库单 -> 采购发票 -> 付款单 -> 应付看板 / 应付账龄 / 资金流水
```

讲解重点：

- 入库单是应付核对的重要来源。
- 采购发票用于确认正式开票金额。
- 付款单记录实际付款。
- 应付看板展示已付、待付和逾期风险。

## 例外财务单据

系统还支持：

- 客户退款单。
- 供应商退款单。
- 费用单。
- 报销单。
- 财务调整单。

这些单据用于处理退款、运杂费、银行手续费、员工报销、尾差、折让、汇差和坏账等例外事项。

## 报表和核对

财务应重点使用：

- 应收看板。
- 应付看板。
- 应收账龄。
- 应付账龄。
- 资金流水。
- 发票差异看板。

## 注意事项

- 财务单据应关联对应合同、入库单或出库单。
- 收付款会影响资金流水。
- 发票金额应与入库单、出库单金额核对差异。
- 退款、费用和调整会影响合同余额和经营判断。
