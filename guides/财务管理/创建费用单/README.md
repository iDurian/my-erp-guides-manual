# 如何创建费用单

本指引用于培训新用户登记物流费、报关费、港杂费、银行手续费等实际出账费用。示例采用采购合同相关的拖车费和报关费，覆盖查找费用归集来源、新建费用单、关联采购合同、填写往来单位、选择实际出账账户、填写多行费用明细、保存确认、查看资金流水，以及回到费用单复核来源和明细。

## 适用场景

- 支付物流、拖车、海运、报关、港杂、仓储、文件、保险等费用。
- 支付银行手续费、平台服务费或其他杂费。
- 一笔费用需要归集到销售合同或采购合同。
- 财务需要登记真实现金流出，并按公司账户进入资金流水。

## 前置条件

- 已确认费用账单、付款对象、付款金额和币种。
- 已确认实际出账公司账户。
- 如果费用需要归集到合同，已明确对应销售合同或采购合同。
- 多项费用合并付款时，已拆分每项费用类别和金额。

## 字段填写说明

| 字段 | 是否必填 | 填写方式 | 影响 |
|---|---|---|---|
| 往来单位 | 必填 | 选择物流公司、报关行、银行、服务商或相关客户/供应商 | 资金流水和费用追溯的付款对象 |
| 来源单号 | 可选 | 选择销售合同或采购合同 | 用于说明费用归属来源 |
| 关联合同 | 可选/建议 | 选择要归集费用的合同 | 便于合同维度核对实际费用 |
| 币种 | 必填 | 按费用账单或银行出账币种选择 | 决定金额展示和资金流水币种 |
| 出账账户 | 必填 | 选择实际付款的公司账户 | 保存确认后按该账户进入资金流水 |
| 费用类别 | 必填 | 运费、报关费、港杂费、银行手续费等 | 影响费用分类和后续统计 |
| 摘要 | 必填 | 简短写明费用内容 | 列表、资金流水和导出中快速识别 |
| 说明 | 可选 | 写柜型、票数、服务范围、账单号等 | 便于复核费用明细 |
| 数量 | 必填 | 按柜、票、项、次等填写 | 参与费用金额计算 |
| 单位 | 建议填写 | 柜、票、项、次等 | 便于理解费用口径 |
| 单价/金额 | 必填 | 按实际费用账单填写 | 本行费用金额 |
| 备注 | 建议填写 | 写付款依据、合同来源和特殊说明 | 便于财务、采购、审计追溯 |
| 保存状态 | 必填 | 草稿 / 已确认 | 已确认后进入资金流水 |

关键规则：

```text
费用单 = 实际费用出账事实
关联合同 = 费用归集和追溯口径
资金流水 = 只统计已确认的真实现金事件
```

## 步骤 01：找到需要归集费用的采购合同

![步骤 01：找到需要归集费用的采购合同](./screenshots/01-找到需要归集费用的采购合同.png)

进入“采购管理 > 采购合同”，找到需要归集物流、报关或港杂费用的采购合同。

## 步骤 02：打开采购合同详情

![步骤 02：打开采购合同详情](./screenshots/02-打开采购合同详情.png)

打开采购合同详情，核对供应商、合同金额和业务背景，确认费用应归集到这笔采购。

## 步骤 03：确认费用归集依据

![步骤 03：确认费用归集依据](./screenshots/03-确认费用归集依据.png)

费用单不改变入库数量，但可以关联合同，方便后续按合同核对实际费用。

## 步骤 04：进入费用单列表

![步骤 04：进入费用单列表](./screenshots/04-进入费用单列表.png)

进入“财务管理 > 费用单”。费用单用于登记物流、报关、港杂、银行手续费等实际出账。

## 步骤 05：新建费用单草稿

![步骤 05：新建费用单草稿](./screenshots/05-新建费用单草稿.png)

点击“新增费用单”后进入草稿。草稿不会进入资金流水。

## 步骤 06：关联采购合同并填写往来单位

![步骤 06：关联采购合同并填写往来单位](./screenshots/06-关联采购合同并填写往来单位.png)

先选择实际付款对象，再选择来源合同和关联合同。费用单的往来单位通常是物流公司、报关行或银行，不一定是采购合同供应商。

## 步骤 07：核对费用单基本信息

![步骤 07：核对费用单基本信息](./screenshots/07-核对费用单基本信息.png)

核对往来单位、日期、负责人和币种。费用单可关联合同，但费用支付对象通常不是原供应商。

## 步骤 08：选择实际出账公司账户

![步骤 08：选择实际出账公司账户](./screenshots/08-选择实际出账公司账户.png)

费用单必须选择实际出账公司账户，保存确认后资金流水按这个账户记录现金流出。

## 步骤 09：填写第一行运费

![步骤 09：填写第一行运费](./screenshots/09-填写第一行运费.png)

选择费用类别，填写摘要、说明、数量、单位和单价/金额。费用金额按实际账单填写。

## 步骤 10：新增并填写报关费

![步骤 10：新增并填写报关费](./screenshots/10-新增并填写报关费.png)

同一张费用单可以登记多行费用，适合一次付款包含多项费用的场景。

## 步骤 11：核对费用合计

![步骤 11：核对费用合计](./screenshots/11-核对费用合计.png)

核对每行费用和总金额。本例运费 CNY 650.00、报关费 CNY 280.00，合计 CNY 930.00。

## 步骤 12：填写备注并准备保存

![步骤 12：填写备注并准备保存](./screenshots/12-填写备注并准备保存.png)

备注写清费用来源、关联合同、费用内容、实际出账金额和付款依据。

备注示例：

```text
关联采购合同 CGHT2606289301。本次支付拖车费和报关费合计 CNY 930.00，按实际出账账户登记。
```

## 步骤 13：选择保存状态

![步骤 13：选择保存状态](./screenshots/13-选择保存状态.png)

金额或账户未确认时保存到草稿；费用已经实际出账时选择“保存并已确认”。

状态说明：

| 状态 | 适用情况 | 后续影响 |
|---|---|---|
| 保存到草稿 | 费用金额、账户或审批仍需确认 | 不进入资金流水 |
| 保存并已确认 | 费用已经实际付款 | 进入资金流水，作为现金流出 |

## 步骤 14：保存并确认费用单

![步骤 14：保存并确认费用单](./screenshots/14-保存并确认费用单.png)

保存并已确认后，费用单成为实际现金流出，会进入资金流水。

## 步骤 15：回到费用单列表验证

![步骤 15：回到费用单列表验证](./screenshots/15-回到费用单列表验证.png)

回到费用单列表搜索单号，确认费用单已经生成并处于已确认状态。

## 步骤 16：查看资金流水出账记录

![步骤 16：查看资金流水出账记录](./screenshots/16-查看资金流水出账记录.png)

资金流水会记录已确认费用单的现金流出，可按账户和日期核对费用付款。

## 步骤 17：复核费用单来源和明细

![步骤 17：复核费用单来源和明细](./screenshots/17-复核费用单来源和明细.png)

再次打开费用单，复核来源合同、关联合同和费用明细，确保费用归集可追溯。

## 常见错误

- 未选择实际出账公司账户，导致无法保存确认。
- 费用已经实际付款，但只保存到草稿，导致资金流水缺失。
- 费用类别全部填“其他”，后续难以按费用类型统计。
- 多项费用合并付款时没有拆行，导致运费、报关费和手续费混在一起。
- 费用需要归集到合同，但没有选择来源单号或关联合同。
- 往来单位误填为合同供应商，实际付款对象是物流公司或报关行。
- 金额和银行实际出账不一致，备注也没有说明差异。
- 币种和实际出账账户币种不一致。

## 保存前检查清单

- 往来单位是实际收款方。
- 来源单号和关联合同已按费用归属选择。
- 实际出账公司账户已选择。
- 费用类别、摘要、说明、数量、单位和金额已核对。
- 合计金额与费用账单、银行出账金额一致。
- 多项费用已拆分为多行。
- 备注已写清费用来源、合同、付款依据和特殊说明。
- 如要形成真实现金流出，保存时选择“保存并已确认”。
