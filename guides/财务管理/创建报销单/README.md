# 如何创建报销单

本指引用于培训财务用户登记员工报销。示例采用员工线下提交差旅费、交通费和办公用品票据，覆盖进入报销单列表、新建草稿、填写报销人和基本信息、拆分报销费用明细、选择公司付款账户、填写线下确认备注、查看附件区、保存为已确认、列表验证、资金流水验证，以及回到报销单复核明细。

## 适用场景

- 员工已垫付差旅、交通、住宿、办公用品、样品寄送等费用。
- 员工已在线下提交发票、收据、付款截图或审批记录。
- 财务需要记录公司实际向员工付款的现金流出。
- 报销费用需要按类别拆分，便于后续统计和复核。
- 某笔报销与销售合同或采购合同相关时，可以选择来源单号或关联合同做追溯。

## 前置条件

- 已确认报销人、报销金额、币种和付款日期。
- 已完成线下票据核对和主管确认。
- 已确认实际付款的公司账户。
- 多项费用合并报销时，已拆分每项费用类别和金额。
- 如果报销需要归集到合同，已明确对应销售合同或采购合同。

## 字段填写说明

| 字段 | 是否必填 | 填写方式 | 影响 |
|---|---|---|---|
| 报销人 | 必填 | 填写实际垫付员工姓名 | 列表、资金流水和后续追溯的往来对象 |
| 单据日期 | 必填 | 填写报销登记日期 | 报销单列表和统计日期 |
| 付款日期 | 必填 | 填写公司实际付款或计划付款日期 | 用于判断和核对付款时间 |
| 来源单号 | 可选 | 选择相关销售合同或采购合同 | 说明报销来源，不选也可保存 |
| 关联合同 | 可选 | 选择需要归集费用的合同 | 用于合同维度追溯报销费用 |
| 币种 | 必填 | 按报销和付款币种选择 | 决定金额展示和资金流水币种 |
| 公司付款账户 | 必填 | 选择实际出账公司账户 | 已确认后按该账户进入资金流水 |
| 费用类别 | 必填 | 差旅费、交通费、住宿费、办公用品、员工报销等 | 影响费用分类统计 |
| 摘要 | 必填 | 简短写明费用内容 | 列表和明细中快速识别 |
| 说明 | 可选 | 写行程、票据范围、用途、次数等 | 便于复核费用合理性 |
| 数量 | 必填 | 按次、项、批、晚等填写 | 参与金额计算 |
| 单位 | 建议填写 | 次、项、批、晚等 | 便于理解费用口径 |
| 单价/金额 | 必填 | 按票据和审批结果填写 | 本行报销金额 |
| 行备注 | 建议填写 | 写票据、截图、用途等说明 | 便于逐行追溯 |
| 线下确认备注 | 建议填写 | 写票据已核对、主管已确认、付款说明 | 便于财务和审计追溯 |
| 附件 | 建议上传 | 上传发票、收据、付款截图、审批记录 | 支撑报销真实性和后续复核 |
| 保存状态 | 必填 | 草稿 / 已确认 | 草稿不进资金流水；已确认表示已付款并进入资金流水 |

关键规则：

```text
报销单 = 员工垫付款项的报销登记
草稿 = 待核对或待付款，不进入资金流水
已确认 = 财务已付款，按公司账户形成现金流出
```

## 步骤 01：进入报销单列表

![步骤 01：进入报销单列表](./screenshots/01-进入报销单列表.png)

进入“财务管理 > 报销单”，先查看当前报销记录、搜索区和“新增报销单”入口。

## 步骤 02：新建报销单草稿

![步骤 02：新建报销单草稿](./screenshots/02-新建报销单草稿.png)

点击“新增报销单”后进入草稿。草稿只保存填写内容，不代表公司已经付款。

## 步骤 03：填写报销人和基本信息

![步骤 03：填写报销人和基本信息](./screenshots/03-填写报销人和基本信息.png)

填写报销人、单据日期、付款日期、负责人、部门和币种。报销人应填写实际垫付员工。

## 步骤 04：确认来源和关联合同可选

![步骤 04：确认来源和关联合同可选](./screenshots/04-确认来源和关联合同可选.png)

来源单号和关联合同不是必填。普通员工日常报销可以留空；如果费用明确属于某个销售合同或采购合同，再选择对应合同。

## 步骤 05：填写第一行差旅费

![步骤 05：填写第一行差旅费](./screenshots/05-填写第一行差旅费.png)

第一行填写差旅费，录入摘要、说明、数量、单位、金额和行备注。金额按票据和审批结果填写。

## 步骤 06：新增并填写交通费

![步骤 06：新增并填写交通费](./screenshots/06-新增并填写交通费.png)

点击“新增费用行”，把交通费单独拆一行填写。同一张报销单可以登记多项费用。

## 步骤 07：新增并填写办公用品

![步骤 07：新增并填写办公用品](./screenshots/07-新增并填写办公用品.png)

继续新增办公用品费用。不要把差旅、交通、办公用品全部合并成一行“其他”，否则后续统计和复核会变困难。

## 步骤 08：核对报销合计

![步骤 08：核对报销合计](./screenshots/08-核对报销合计.png)

核对每行金额和总金额。本例差旅费 CNY 860.00、交通费 CNY 135.00、办公用品 CNY 128.00，合计 CNY 1,123.00。

## 步骤 09：选择公司付款账户

![步骤 09：选择公司付款账户](./screenshots/09-选择公司付款账户.png)

选择实际付款的公司账户。报销单必须选择公司付款账户；保存为已确认后，资金流水按该账户记录现金流出。

## 步骤 10：填写线下确认备注

![步骤 10：填写线下确认备注](./screenshots/10-填写线下确认备注.png)

线下确认备注要写清票据、审批和付款依据。

备注示例：

```text
已线下核对发票、行程单和主管确认记录。本次报销合计 CNY 1,123.00，财务已按人民币主付款账户付款。
```

## 步骤 11：查看报销附件区

![步骤 11：查看报销附件区](./screenshots/11-查看报销附件区.png)

报销附件可上传发票、收据、付款截图和线下确认材料。培训时建议说明：附件不是金额计算来源，但用于证明报销真实、完整、可追溯。

## 步骤 12：准备保存报销单

![步骤 12：准备保存报销单](./screenshots/12-准备保存报销单.png)

保存前回到顶部，确认报销人、金额、公司付款账户、线下确认备注和附件情况。

## 步骤 13：选择保存状态

![步骤 13：选择保存状态](./screenshots/13-选择保存状态.png)

金额、票据或审批仍未确认时保存草稿；如果公司已经实际付款，选择“保存并已确认”。

状态说明：

| 状态 | 适用情况 | 后续影响 |
|---|---|---|
| 保存到草稿 | 票据、审批、金额或付款账户仍需核对 | 不进入资金流水 |
| 保存并已确认 | 财务已经付款或确认作为正式付款事实 | 进入资金流水，作为现金流出 |

## 步骤 14：保存并确认报销单

![步骤 14：保存并确认报销单](./screenshots/14-保存并确认报销单.png)

保存并已确认后，报销单成为正式财务事实。系统将显示单据编号、已确认状态、报销人和报销金额。

## 步骤 15：回到报销单列表验证

![步骤 15：回到报销单列表验证](./screenshots/15-回到报销单列表验证.png)

回到报销单列表搜索单号，确认报销单已经生成、状态为已确认、金额和报销人正确。

## 步骤 16：查看资金流水出账记录

![步骤 16：查看资金流水出账记录](./screenshots/16-查看资金流水出账记录.png)

资金流水中会出现已确认报销单，按公司账户记录 CNY 1,123.00 的现金流出。

## 步骤 17：复核报销单明细

![步骤 17：复核报销单明细](./screenshots/17-复核报销单明细.png)

再次打开报销单，复核报销人、付款账户、线下确认备注和三行费用明细，确保单据可以被财务和审计追溯。

## 常见错误

- 报销人填写成客户或供应商名称，而不是实际垫付员工。
- 公司已经付款，但只保存为草稿，导致资金流水缺失。
- 未选择公司付款账户，无法形成准确现金流。
- 多项费用合并成一行“员工报销”或“其他”，后续难以按类别统计。
- 金额与票据、审批或实际付款不一致，备注中也没有说明差异。
- 付款日期和实际付款日期不一致，影响月度资金核对。
- 报销涉及合同成本，但没有选择来源单号或关联合同。
- 未上传或未备注线下确认材料，后续审计难以复核。

## 保存前检查清单

- 报销人是实际垫付员工。
- 单据日期、付款日期和币种已核对。
- 如需归集到合同，来源单号或关联合同已选择。
- 公司付款账户是实际出账账户。
- 每行费用类别、摘要、说明、数量、单位和金额已填写。
- 合计金额与票据、审批和实际付款一致。
- 多项费用已拆分成多行。
- 线下确认备注已写清票据、审批和付款依据。
- 如已经付款，保存时选择“保存并已确认”。
