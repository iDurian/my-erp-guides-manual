# 任务级业务流程教程清单

本文用于跟踪“细分到具体任务”的截图教程制作。模块级 guide 说明系统有哪些入口；任务级 guide 说明某一件事如何一步一步完成、每个字段怎么填、保存后如何验证。

## 制作规范

- 目录使用中文命名，例如 `guides/基础资料/创建新客户/`。
- 每个任务目录包含 `README.md` 和 `screenshots/`。
- 每张截图包含步骤序号、红框标注和底部字幕。
- `README.md` 必须包含适用场景、字段说明、步骤截图、常见错误和保存前检查清单。
- 截图使用演示数据，不暴露真实客户、密码、服务器地址或密钥。
